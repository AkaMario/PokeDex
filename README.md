## Creación de la cuenta en la nube elegida

En este proyecto no fue necesario crear una cuenta nueva en la nube, ya que la cuenta de Microsoft Azure ya se encontraba creada previamente.

Por lo tanto, para realizar el despliegue únicamente se hizo lo siguiente:

1. Se ingresó al portal de Microsoft Azure.
2. Se inició sesión con la cuenta ya existente.
3. Se accedió al panel principal de Azure.
4. Desde allí se utilizó el servicio necesario para desplegar la aplicación.
